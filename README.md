***Key34***   
Проект Перехватывает нажатие клавиши 34 (чтобы узнать о ней Подробнее, загуглите key rule 34) и автоматически заменяет её на русскую заглавную букву П.  
Программа работает в фоне и останавливается вручную (закрытием).
  
**Требования**  
Необходима библиотека keyboard для Перехвата и эмуляции нажатий клавиш:     
`pip install keyboard`  
    
**ЗаПуск**  
ЗаПуск осуществляется через файл start.bat (желательно от имени администратора)
